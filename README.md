## Installation

1. Clone this repository:
   ```sh
   git clone https://github.com/CrypticEnigmaX/weather_app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd weather_app
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
